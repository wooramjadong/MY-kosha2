# 🚨 사고발생 사이렌 포스터 메이커 (Accident Siren Poster Maker)

산업안전보건공단 스타일의 안전 포스터를 손쉽게 제작할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능
- **AI 이미지 생성**: OpenAI DALL·E 3를 이용해 사고 상황에 맞는 이미지를 자동 생성합니다. (API 키 필요)
- **데모 모드**: API 키 없이도 내장된 예시 이미지를 통해 기능을 체험할 수 있습니다.
- **이미지 업로드**: 사용자가 직접 이미지를 업로드하여 포스터를 만들 수 있습니다.
- **포스터 다운로드**: 완성된 포스터를 고화질 PNG 이미지로 저장할 수 있습니다.
- **자동 저장**: 입력한 API 키는 브라우저에 안전하게 저장되어 재입력이 필요 없습니다.
